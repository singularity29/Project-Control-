# Project Control
A Chromium browser extension designed to strip UI structural distractions from YouTube and Instagram. Integrates a client-side AI model (NSFWJS) for dynamic content blurring.

### Manual Installation Protocol
1. Click the green **Code** button and select **Download ZIP**.
2. Extract the downloaded `.zip` archive to a local directory.
3. Open a Chromium-based browser (Chrome, Brave, Edge) and navigate to the extensions dashboard (`chrome://extensions/`).
4. Toggle **Developer mode** to ON (located in the top-right corner).
5. Click **Load unpacked** (top-left corner) and select the extracted `Project-Control-main` folder.
